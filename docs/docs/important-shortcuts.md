# ⏩ Important shortcuts

| Shortcut      | Action                           |
| ------------- | -------------------------------- |
| `CMD+L`       | New chat                         |
| `CMD+SHIFT+R` | Bring last terminal text to chat |
| `CMD+'`       | Switch models                    |
| `CMD+[`       | Big                              |
| `CMD+0`       | Previous                         |
| `CMD+O`       | History                          |
| `CMD+;`       | Close                            |
| `CMD+SHIFT+L` | Append selected                  |

All shortcuts can be seen within PearAI itself by `CMD+SHIFT+P` → `Keyboard shortcuts`
